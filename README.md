## ⚡ Categorias

| Categoria | Descrição |
|---|---|
| `XSS` | Cross-Site Scripting — reflected, stored, DOM |
| `SQLi` | SQL Injection — error-based, blind, time-based |
| `IDOR` | Insecure Direct Object Reference |
| `SSRF` | Server-Side Request Forgery |
| `Open-Redirect` | Redirecionamento não validado |
| `Auth-Bypass` | Bypass de autenticação e autorização |
| `LFI-RFI` | Local/Remote File Inclusion |
| `Misc` | Outros payloads úteis |

---

## 🛠️ Ferramentas que uso

- `Burp Suite` — interceptação e manipulação de requisições
- `ffuf` / `dirsearch` — fuzzing de endpoints
- `curl` — testes manuais de API
- `nuclei` — templates de vulnerabilidades
